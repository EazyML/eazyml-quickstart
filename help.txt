-h, --help                  :   Help
-u, --username              :   EazyML username
-p, --api_key               :   EazyML api_key. User can download it from UI (MyAccount->API)
-x, --prefix_name           :   Prefix to be used while saving response.
-g, --config_file           :   Configurations file to override the default EazyML configurations.
-f, --train_file            :   Train filename with absolute path.
-o, --outcome               :   Outcome column name. It must be provided if train filename is provided.
-i, --id_col                :   Id col in train data.
-c, --discard_col_list      :   Comma separated list of columns which user wants to discard in train data.
-u, --impute                :   Pass --impute if want to do imputation, default is "no".
-l, --outliers              :   Pass --outlier if want to remove outliers, default is "no".
-d, --dataset_id            :   Dataset upload id on which user wants to build model/s.
-v, --predictive            :   Builds predictive model.
-a, --augi                  :   Builds augmented intelligence model.
-m, --model_id              :   Pass model_id to perform prediction.
-r, --predict_file          :   Prediction filename with absolute path.
-t, --prediction_dataset_id :   Prediction dataset id to understand the predictions.
-e, --explain_rec_nums      :   Comma separated list of record numbers user want to understand.
-s, --clean_resources       :   Pass --clean_resources to clean resources else data upload or models will not be cleaned.
